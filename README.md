# E-Commerce Delivery Analytics Dashboard

## Overview

This project analyzes a **700,000+ order** e-commerce delivery dataset to uncover insights into sales performance, financial metrics, delivery operations, and customer returns. The data was cleaned using **Python**, and an interactive dashboard was developed in **Power BI**.

## Tools Used

* Python
* Pandas
* Power BI
* DAX
* Jupyter Notebook

## Key Features

* Data Cleaning & Preprocessing
* Executive Overview Dashboard
* Financial Analysis
* Return Analysis
* Delivery Analysis
* Interactive KPIs & Visualizations

## Key Insights

* 700,000+ Orders Processed
* USD 3.15 Billion Gross Revenue
* USD 2.33 Billion Net Revenue
* 4.30% Month-over-Month Revenue Growth
* 7.94% Return Rate
* 90% Successful Deliveries
* 5.50 Days Average Fulfillment Time

## Project Outcome

This dashboard provides actionable business insights to help monitor revenue, delivery performance, customer returns, and overall operational efficiency.
